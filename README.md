# Premium Digital-Analogue Clock

A modern, stylish *premium clock* that combines *analogue and digital displays* in a single interface. This project was *entirely designed and coded by NAKUL*, using original ideas and implementation.

## Features

- *Dual Display:* Analogue clock hands + digital time display.
- *Real-Time Functionality:* Accurate time updates every second.
- *Premium Design:* Sleek, modern interface with smooth animations.
- *Mobile Friendly:* Works seamlessly on desktops, tablets, and mobile devices.
- *Customizable Appearance:* Easy to modify colors, fonts, and styles.
